# PSX 0.9.2 修正内容

- 登録フォルダ移動時の `ClearHistory` と最大3,000件の `AddToHistory` 再投入を廃止しました。
- カレントディレクトリの再判定を省略し、設定探索結果をディレクトリ単位でキャッシュします。
- Enterキー処理から毎回のコンテキスト探索を外し、プロンプト表示時の移動検出に統一しました。
- 履歴は前後空白を除去した完全一致で重複を除き、新しい側を残して最大1,000件に整理します。
- 履歴日時メタデータも、現在残っているコマンドに合わせて整理します。
- 公開コマンドを `psx` のみに統一し、`psctx`、`psprojhist`、`pph` のラッパーを削除しました。
- 既存の `.psctx.json`、`.psctx.ps1`、`.pslocal` 履歴との互換性は維持しています。

## 導入

```powershell
.\Install-PSX.ps1
. $PROFILE
psx --version
```
